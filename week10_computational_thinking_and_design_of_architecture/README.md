## Homework 0: architecture design of the final project
* Design the architecture of your final project and document it in a diagram. 
* Push the diagram and also some text in your project's `README.md` that 
descripe the architecture design of your final project.

No matter what tools you use, the uploaded figure(s) should be a file
in common figure format, like jpg, png, etc. And the file itself can
be put in the `inputs/` folder of the package, and referred and shown in 
the `README.md` file. You can check the README file [here](https://github.com/DTUWind-46120-2026/46120-PiWE/blob/main/week08_final_project_intro/wind_turbine_modeling/README.md) for an example of how to do this.

Note that the architecture you design for now can be up to changes when you 
progress with your final project, don’t treat it as final and fixed.
